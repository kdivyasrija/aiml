# Financial Fraud Detection using Machine Learning

## Overview

This repository contains a machine learning project focused on detecting fraudulent financial transactions. The notebook demonstrates the complete workflow of a data science project including data preprocessing, model training, hyperparameter tuning, evaluation, and performance comparison.

The project uses multiple classification algorithms to identify fraudulent transactions and compares their performance to determine the most effective model.

## Features

* Data preprocessing and feature selection
* Train-test data splitting
* Implementation of multiple machine learning models
* Hyperparameter tuning for model optimization
* Model evaluation using performance metrics
* Visual comparison of model results

## Machine Learning Models Used

The following models are implemented and evaluated:

* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Random Forest

## Workflow

1. Import required libraries
2. Load and explore the financial transaction dataset
3. Separate features and target variable
4. Split data into training and testing sets
5. Train multiple machine learning models
6. Make predictions on test data
7. Evaluate model performance
8. Perform hyperparameter tuning
9. Compare model performance using visualization
